JsSkal

A JavaScript Version of Pyskal

What is it?

JsSkal is a client-side web page that is able to transform text into a image, in a non-reversible process.


How does it work? (For Nerds)

First, a hash algorithm is used, hashing the text input into a hex string.

The hash is then used to set each pixel color, creating the image.
